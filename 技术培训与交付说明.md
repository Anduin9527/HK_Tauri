# 技术培训与交付说明（简版）

面向对象：甲方技术人员  
目标：便于接手维护、部署与二次开发

---

## 1. 系统概述
本系统为本地部署的工业视觉缺陷检测软件，整体由三部分构成：  
- 桌面端界面（Tauri + React）  
- 本机后端服务（Python FastAPI，默认 127.0.0.1:8000）  
- 模型推理（Ultralytics YOLO，CPU 推理）

---

## 2. 技术实现方式与架构
### 2.1 进程结构
- 桌面端启动后，自动拉起后端 sidecar 进程
- 前端通过 HTTP + Socket.IO 与后端通信
- 后端负责相机接入、图像采集、推理、日志与截图落盘

### 2.2 目录与数据落盘
- 运行时数据目录：HK_Tauri_Data  
- 历史记录：HK_Tauri_Data/history/  
- 配置文件：HK_Tauri_Data/config.json  
- 后端日志：HK_Tauri_Data/backend.log  

可通过接口 `/paths` 查看实际落盘路径。

### 2.3 相机接入
- 默认使用海康 MVS SDK（MvImport），支持 GigE / USB 工业相机
- 支持 4 路逻辑槽位（Slot 0~3）
- 支持曝光、增益参数下发（按 Slot 保存配置）

---

## 3. 数据集与模型训练
数据采集：
1. 海康威视工业采集软件采集图像  
2. Roboflow 线上标注  
3. YOLO 训练得到训练模型  

模型加载策略：
- 默认加载 `backend/models/best.onnx` 或 `backend/models/best.pt`
- 支持 `yolo26s / yolo26n` 等命名模型
- 可在界面设置模型名、类型、置信度、推理尺寸

---

## 4. 核心功能实现过程
### 4.1 运行流程
1. 启动 Tauri 桌面端  
2. 后端服务自动启动  
3. 发现相机 → 绑定 Slot → 开始视频流  
4. 自动检测或手动触发检测  
5. 生成日志与抓拍结果图

### 4.2 推理与告警
- 自动模式：持续推理，检测到异常写入日志并存图  
- 手动模式：点击“单次检测”对当前帧执行推理  
- 图片调试：上传图片进行离线检测

### 4.3 日志与历史记录
- 实时日志通过 Socket.IO 推送  
- 历史日志写入 `events.log`  
- 抓拍结果保存在 `history` 目录

## 5. 依赖环境与软件平台（详细安装教程）
以下步骤以 Windows 为例，路径均为示例，可按实际安装位置调整。

### 5.1 基础环境
1. 安装 Python 3.9+（建议 3.10）
   - 官网：https://www.python.org/downloads/  
   - 安装时勾选 “Add python.exe to PATH”

2. 安装 Node.js 16+（建议 18 LTS）
   - 官网：https://nodejs.org/  
   - 安装完成后验证：
```powershell
node -v
npm -v
```

3. 安装 Rust（Tauri 构建需要）
   - 官网：https://www.rust-lang.org/tools/install  
   - 安装完成后验证：
```powershell
rustc -V
cargo -V
```

4. 安装海康 MVS SDK Runtime（工业相机）
   - 安装后确保环境变量 `MVCAM_COMMON_RUNENV` 存在  
   - 需重启终端或系统后生效  

---

### 5.2 后端依赖安装（Python）
进入后端目录：
```powershell
cd c:\Users\Administrator\Desktop\HK_Tauri-main\backend
```

创建虚拟环境：
```powershell
python -m venv venv
.\venv\Scripts\activate
```

安装依赖：
```powershell
pip install -r requirements.txt
```

检查模型文件：
- 将训练好的模型放入 `backend\models\`  
- 文件名默认识别 `best.onnx` / `best.pt`

启动后端：
```powershell
python main.py
```

验证接口：
```powershell
# 访问浏览器 http://127.0.0.1:8000/status
```

---

### 5.3 前端依赖安装（React + Tauri）
进入前端目录：
```powershell
cd c:\Users\Administrator\Desktop\HK_Tauri-main\frontend
```

安装前端依赖：
```powershell
npm install
```

启动前端开发模式（需后端已运行）：
```powershell
npm run tauri dev
```

---

### 5.4 打包构建（可执行程序）
直接执行shell脚本：
```powershell
.\build_windows.ps1
```

安装包输出目录：`frontend\src-tauri\target\release\bundle\nsis*-setup.exe`


## 6. 关键接口（联调用）
- `GET /cameras/discover`：发现相机  
- `POST /cameras/{slot}/connect`：绑定相机  
- `POST /cameras/{slot}/disconnect`：断开相机  
- `GET /video_feed/{slot}`：视频流  
- `POST /trigger/detect`：手动检测  
- `GET /config/settings`：获取参数  
- `POST /config/settings`：更新参数  
- `GET /logs`：拉取历史日志  
- `POST /predict/image`：图片检测
